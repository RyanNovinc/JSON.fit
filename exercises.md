# JSON.fit Exercise Library

This file is the canonical exercise reference for AI-generated programs on JSON.fit.

## Instructions for AI

When generating workout programs for JSON.fit users, you MUST:

1. Select exercises ONLY from the list below. Do NOT invent exercises not in this library.
2. Use the exact exercise name shown — do not vary or abbreviate names.
3. Use the exact primary and secondary muscle tags shown for each exercise. Do not re-tag based on your own judgment.
4. If a user's equipment or preferences rule out all options for a movement pattern, omit that pattern rather than inventing an exercise.
5. If suggesting alternatives for an exercise, only use exercises listed in this library.

## Muscle Taxonomy

Valid muscle tags (use exactly these names):

Chest, Front Delts, Side Delts, Rear Delts, Lats, Upper Back, Traps, Biceps, Triceps, Forearms, Quads, Hamstrings, Glutes, Calves, Core, Lower Back, Neck, Obliques, Serratus Anterior, Hip Abductors, Hip Adductors, Shins

## Tagging Philosophy

Primary = main driver through meaningful range of motion under load.
Secondary = assists through meaningful ROM under load but is not the main driver.

Muscles that only stabilize or isometrically contract are NOT tagged as secondary. This is why squats don't tag Core, pulls don't tag Forearms, and overhead presses don't tag Core despite all three muscles contracting during those lifts.
