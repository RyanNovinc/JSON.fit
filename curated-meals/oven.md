# Curated Meals — Oven

3 meals that can be cooked in an oven. All are multi-method (also available via slow cooker) — only the oven method is shown here.

For multi-method meals, the same plates are available regardless of which method cooked the base.

Method equipment requirements are listed per method. The user must have ALL listed equipment. If they don't, skip the method (try another method of the same meal, or a different meal).

---

## pulled_pork — Slow-Cooked BBQ Pulled Pork

- Primary protein: pork
- Eligible slots: lunch, dinner, second_lunch, early_dinner
- Scale: 0.7–1.5
- Produces servings: 8
- Allergens: Dairy, Eggs, Gluten/Wheat, Soy, Fish
- Also available in: slow-cooker.md

- Equipment: oven, stovetop
- Active 30 min, total 360 min, skill 2

- `baked_potato` — Loaded Pulled Pork Baked Potato

  - Plate equipment: oven
  - Base servings consumed: 1.0

  - Macros: 1155 kcal / 61g P / 55g C / 74g F / 10g fiber
