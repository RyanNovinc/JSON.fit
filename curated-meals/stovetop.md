# Curated Meals — Stovetop

5 meals that can be cooked stovetop. Some are multi-method (also available via slow cooker, oven, or pressure cooker) — only the stovetop method is shown here.

For multi-method meals, the same plates are available regardless of which method cooked the base.

Method equipment requirements are listed per method. The user must have ALL listed equipment. If they don't, skip the method (try another method of the same meal, or a different meal).

---

## butter_chicken — Butter Chicken
- Cuisine: indian
- Primary protein: chicken
- Eligible slots: lunch, dinner, second_lunch, early_dinner
- Scale: 0.7–1.5
- Produces servings: 6
- Allergens: Dairy
- Also available in: slow-cooker.md, pressure-cooker.md

### Stovetop scratch method (`stovetop_scratch`)
- Equipment: stovetop
- Active 25 min, total 35 min, skill 3, scratch

### Stovetop shortcut method (`stovetop_shortcut`)
- Equipment: stovetop
- Active 12 min, total 20 min, skill 2, uses Patak's jar sauce

### Plates

  - Macros: 720 kcal / 48g P / 65g C / 28g F / 4g fiber
  - Assembly: 0 min
  - Plate equipment: stovetop (for rice)
  - Base servings consumed: 1.0
