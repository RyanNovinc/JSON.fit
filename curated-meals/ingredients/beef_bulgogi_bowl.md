## Plates
### `standard` — Beef Bulgogi over Jasmine Rice
- **Macros:** 820 kcal / 50g P / 96g C / 22g F / 3g fiber
- **Base servings consumed:** 1.0
- **Additional ingredients:** none
- **Note:** Marinate 30 min–overnight. Freeze beef ~1hr for easier thin slicing. Sear in batches so it caramelises, not stews. Pear/apple is an optional tenderiser (or omit). 200g beef/portion; jasmine rice for bulking carbs. Store beef separate from rice; keeps 3-4 days. Optional add-ons (not in base macros): fried egg, kimchi, pickled cucumber.
