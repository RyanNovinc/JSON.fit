# Curated Meals — Blender

12 bulking smoothies. Single-method blender. Single-plate. Each is one serving.

All smoothies share:
- `produces_servings`: 1
- `Scale`: 0.6–1.3
- Method: `blender`, active 4 min, total 5 min, skill 1
- Plate id: `standard`, assembly 0 min, `Plate equipment: none`, `Base servings consumed: 1.0`
- Default eligible slots: breakfast, brunch, morning_snack, afternoon_snack, pre_workout, post_workout

Where a smoothie's entry includes an `Eligible slots:` line, that line is the COMPLETE allowed set for that meal and replaces the default above — read it as "exactly these," not "the defaults plus these."
