How I got to the start line.

The scan history above starts at 77 kg on 1 July 2026. Here is the decade that came before it, so you know exactly who is doing this bulk, and why.

185 cm and under 60 kg right through my teens. 67 kg when I left uni. Then I did a first bulk up to 82 kg over about 18 months, and it worked. But I did it the miserable way, the same meals every day, forcing it down, so it never stuck, and I drifted back to 77 kg. That is the whole reason I built JSON.fit, and the reason this run is tracked, sustainable, and built to actually last. Same size I proved I could reach, done properly this time.
