How I got to the start line.

The record above starts at 77 kg on 1 July 2026. Here is the decade that came before it, so you know exactly who is doing this bulk, and why.

185 cm and under 60 kg right through my teens. 67 kg leaving uni. A first bulk got me to 82 kg in 18 months, but I did it the miserable way, the same meals every day, forced down, so it never stuck and I drifted back to 77 kg. That's why JSON.fit exists, and why this run is built to last. Same size I've already proven I can reach, done properly this time.
